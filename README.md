# finfmt

`finfmt` is an experimental Rust library for encoding and decoding financial
wire formats such as ISO 8583-style bitmap messages, fixed-width records, and
BER-TLV fields.

## Status

This crate is in early development and is being published only to reserve the
name and make the work visible. The API, macro syntax, examples, and
documentation are still changing.

Please do not use this crate for production systems yet.
